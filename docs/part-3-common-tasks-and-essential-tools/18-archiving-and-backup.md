# 18. 存档和备份

计算机系统管理员的一个主要任务之一，是保持系统的数据安全。完成这项任务的一个途径是定时执行系统文件的备份。即便你不是一个系统管理员，在设备之间制作拷贝和移动巨大的文件集，也是很有用的。

本章中，我们将看到几个用来管理文件集的常用程序。它们是文件压缩程序：

- gzip - 压缩或扩展文件
- bzip2 - 块排序文件压缩器

还有存档程序：

- tar - 磁带存档工具
- zip - 大包和压缩文件

和文件同步程序：

- rsync - 远程文件和目录同步
